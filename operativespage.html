Lazarus Group

All Tracked Names

Labyrinth Chollima, HIDDEN COBRA, Gaurdians of Peace, ZINC, NICKEL ACADEMY

    Common Malware

  • MagicRAT, QuiteRAT, CollectionRAT, EarlyRAT, DeimosC2, and TigerRAT
    • Uses Qt framework, which is a GUI programming library, but it doesn't use a GUI
    • Used as a recon tool for initial scans sent through C2
    • It is used against VMWare Horizon platforms and CVE-2022-47966
    • MagicRAT is most notably used in a campaign against U.S., Japanese, and Canadian energy providers

APT37

All Tracked Names

Reaper, ScarCruft, Richochet Chollima, InkySquid, Group123, TEMP.Reaper

    Most Notable Attacks

  • Operation Daybreak, Operation Erebus, Golden Time, Evil New Year, Are you Happy?, FreeMilk, North Korean Human Rights, and Evil New Year 2018

    • Common Malware

  • RUHappy, DogCall, ShutterSpeed, PoorAim, ZumKong, CoralDeck, and SoundWave
    • RUHappy can completely wipe a system and only leave them with words displaying "Are You Happy?". It technically hasn't been proven to have been deployed, but an attack on a South Korean power plant left machines in that state.
    • DogCall, ShutterSpeed, and PoorAim have the goal of keystroke logging, screenshots, and file exploration
    • ZumKong is a credential harvesting tool
    • CoralDeck compresses and extracts files
    • SoundWave takes over machines microphones allowing the attacker to listen into all the conversations

Kimsuky

All Tracked Names

STOLEN PENCIL, Thallium, Black Banshee, Velvet Chollima

Cyber-Skills

    Most Notable Attacks

  • 2014 Korea Hydro & Nuclear Power Co, Operation STOLEN PENCIL (2018), Operation Kabar Cobra (2019), and Operation Smoke Screen

    • Common Tactics

  • ReconShark (BabyShark), RandomQuery, autostart manipulation
    • RandomQuery is mainly used for file enumeration and exfiltration. It has recently gained more uses such as key logging and execution.
    • ReconShark is distributed through microsoft office and has a goal of gaining harware information for malware deployed later designed to attack the weaknesses.
    • Using RDP, they change an apps file association to the infected malware

Andariel

All Tracked Names

Silent Chollima, PLUTONIUM, Onyx Sleet

    Most Notable Attacks

  • Operation Black Mine, Operation GoldenAxe, and Campaign Rifle

    • Common Tactics

  • TigerRAT, LightHand, ValidAlpha, SmallTiger
    • TigerRAT can perform keylogging and screen recording through C2 via a Log4j vulnerability
    • SmallTiger delivered through a DLL and its a downloader from C2. Depending on the attack it will download Mimikatz or ProcDump and can extract webbrowser data
    • LightHand is a backdoor that allows for commands through the command shell. They get system storage information and create and delete files
    • ValidAlpha can run files, lists directory contents, download files, take screenshots, and run commands in the shell

APT38

All Tracked Names

NICKEL GLADSTONE, BeagleBoyz, Bluenoroff, Stardust Chollima

    Most Notable Attacks

  • 2016 Bank of Bangladesh heist, Bancomext, and Banco de Chile

    • Common Tactics

  • RATANKBA, ThreatNeedle, BLINDINGCAN, and COPPERHEDGE
    • RATANKBA is used as a C2 server and also executes tasks. A GUI is used by the attacker and tasks get sent to the server which is then pushed to the backdoor once it connects to the server
    • ThreatNeedle has an installer, loader, and a backdoor that manipulates files, profiles systems, and executes commands
    • BLINDINGCAN can do many things such as create processes, file modification, disk information gathering, upload files and more.
    • COPPERHEDGE is a backdoor trojan that can perform data exfiltration, commands, and system recon.
